# Агрегація тегів

Цей файл містить конфігурацію агрегації тегів.

## Формат

```json
{
  "aggregations": [
    {
      "main_tag": "Назва головного тегу",
      "similar_tags": [
        "Схожий тег 1",
        "Схожий тег 2"
      ]
    }
  ]
}
```

## Використання

Агрегація тегів дозволяє об'єднувати схожі теги під одним головним тегом.

Наприклад, якщо у вас є теги "Past Simple", "Паст Сімпл", "past simple", ви можете створити агрегацію:

```json
{
  "aggregations": [
    {
      "main_tag": "Past Simple",
      "similar_tags": [
        "Паст Сімпл",
        "past simple"
      ]
    }
  ]
}
```

## Керування

Агрегаціями можна керувати через веб-інтерфейс за адресою `/admin/test-tags/aggregations` або вручну редагуючи цей файл.
