# Seed Runs V2 Module

Модуль для керування Seed Runs з використанням Livewire (SPA-подібний інтерфейс).

## Опис

Цей модуль надає альтернативний інтерфейс для керування сидерами з використанням Livewire, що забезпечує SPA-поведінку без перезавантаження сторінки.

## Як встановити

1. Скопіюйте папку `app/Modules/SeedRunsV2` у ваш застосунок.
2. Додайте `App\Modules\SeedRunsV2\SeedRunsV2ServiceProvider::class` у масив `providers` файлу `config/app.php`.
3. Переконайтеся, що Livewire встановлено: `composer require livewire/livewire`
4. За потреби опублікуйте конфіг/шаблони:
   ```bash
   php artisan vendor:publish --tag=seed-runs-v2-config
   php artisan vendor:publish --tag=seed-runs-v2-views
   ```
5. Додайте посилання в адмін-навігацію на маршрут `seed-runs.v2.index`.

## Функціональність

### Невиконані сидери
- Перегляд ієрархічного дерева папок/сидерів
- Виконання окремих сидерів
- Виконання всіх невиконаних сидерів
- Позначення як виконані
- Редагування файлу сидера (модальне вікно)
- Видалення файлу сидера
- Створення нового сидера

### Виконані сидери
- Перегляд ієрархічного дерева папок/сидерів
- Пошук по назві сидера (live search)
- Видалення запису про виконання
- Видалення з пов'язаними питаннями
- Перегляд та редагування файлу

## Конфігурація

Файл `config/seed-runs-v2.php` містить:
- `route_prefix` - URL префікс (за замовчуванням `admin/seed-runs-v2`)
- `middleware` - middleware для маршрутів (за замовчуванням `['web', 'auth.admin']`)

## Маршрути

- `GET /admin/seed-runs-v2` - головна сторінка модуля (маршрут: `seed-runs.v2.index`)
