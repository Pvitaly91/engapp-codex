# Посібник з керування тегами для сторінок та категорій

## Огляд

Додано повнофункціональний інтерфейс керування для прив'язування тегів до сторінок теорії та категорій сторінок.

## Нові сторінки адміністратора

### 1. Управління сторінками
**URL:** `/admin/pages-manage`

Функції:
- Перегляд усіх сторінок теорії з інформацією про категорію та кількість тегів
- Створення нових сторінок з вибором тегів
- Редагування існуючих сторінок та їх тегів
- Видалення сторінок

### 2. Управління категоріями
**URL:** `/admin/page-categories-manage`

Функції:
- Перегляд усіх категорій з інформацією про кількість сторінок та тегів
- Створення нових категорій з вибором тегів
- Редагування існуючих категорій та їх тегів
- Видалення категорій

## Інтерфейс вибору тегів

### Форма редагування сторінки

При створенні або редагуванні сторінки доступна секція "Теги":

```
Теги
┌─────────────────────────────────────────────┐
│ ☐ Present Simple (Tenses)                  │
│ ☑ Past Simple (Tenses)                     │
│ ☑ Future Simple (Tenses)                   │
│ ☐ Modal Verbs (Modals)                     │
│ ☐ Conditionals (Structures)                │
│ ...                                         │
└─────────────────────────────────────────────┘
```

**Особливості:**
- Checkbox для кожного тегу
- Показує назву тегу та його категорію в дужках
- Можна вибрати декілька тегів одночасно
- Адаптивна сітка (2 колонки на мобільних, 3 на десктопі)
- Посилання на створення нового тегу, якщо теги ще не створено

### Форма редагування категорії

Аналогічний інтерфейс вибору тегів доступний при створенні/редагуванні категорій.

## Як використовувати

### Додати теги до сторінки:

1. Перейдіть до `/admin/pages-manage`
2. Натисніть "Редагувати" біля потрібної сторінки
3. Прокрутіть до секції "Теги"
4. Виберіть потрібні теги за допомогою checkbox
5. Натисніть "Зберегти зміни"

### Додати теги до категорії:

1. Перейдіть до `/admin/page-categories-manage`
2. Натисніть "Редагувати" біля потрібної категорії
3. Прокрутіть до секції "Теги"
4. Виберіть потрібні теги за допомогою checkbox
5. Натисніть "Зберегти зміни"

### Створити нову сторінку з тегами:

1. Перейдіть до `/admin/pages-manage`
2. Натисніть "Додати сторінку"
3. Заповніть назву, slug, виберіть категорію
4. Виберіть теги в секції "Теги"
5. Натисніть "Створити сторінку"

## Технічні деталі

### Синхронізація тегів

При збереженні форми:
- Всі вибрані теги прив'язуються до сторінки/категорії
- Всі невибрані теги від'єднуються
- Використовується Laravel метод `sync()` для ефективної синхронізації

### Валідація

- `tags` - необов'язковий масив
- `tags.*` - кожен елемент масиву має існувати в таблиці `tags`

### Відображення

На сторінці списку показується:
- Кількість тегів для кожної сторінки/категорії
- Візуальні badge з кількістю

## Інтеграція

Цей інтерфейс доповнює функціонал `/admin/test-tags`, де можна:
- Переглянути всі сторінки/категорії, пов'язані з конкретним тегом
- Використовувати API для програмного керування тегами

## Посилання

- Управління тегами: `/admin/test-tags`
- Управління сторінками: `/admin/pages-manage`
- Управління категоріями: `/admin/page-categories-manage`
- Створити тег: `/admin/test-tags/create`
